# Modelo basado en LLMs para apoyar la toma de decisiones en el triaje médico

Este repositorio contiene un modelo basado en Large Language Models (LLMs), agentes y Retrieval-Augmented Generation (RAG) diseñado para apoyar la toma de decisiones en el proceso de triaje en servicios de urgencias.

En el repositorio se encuentran los códigos y prompts utilizados para el trabajo y algunas comparaciones realizadas con personal de la salud.
